Contributing
------------

* Fork the project and clone your fork

* Create a feature branch with: `git checkout -b <branch>`

* Add your function
  - Put it in the appropriate file under the `./src` folder.
  - Add a test for the function in the appropriate file under `./test`
  - Fill out the type as best you can.

* Testing
  - run `npm test` to have the eslint, flow, and mocha, poke around.

* push your changes

  - `git push origin <branch>`

* open a pull request
